
# Probably the useful Backend Repo

The project is started to create a useful backend repo using NodeJs and ExpressJs from which you can Pull before building your next big product.




## Feature we cover

- ES6 Code structure, Proper Folder structure with Proper Commenting
- Common Models repo for MongoDB
- TokenBased Login with Encryption
- Cookie based Login with encryption
- Middleware for user authorisation
- Swagger for API documentation
- Middleware and Utils for different unit conversions
- File handling using Multer 
- File handeling using  Amazon S3
- Queuing service RabbitMQ
- Notification Services Email
- Notification Service Message MSG91
- Notification Service Push Notification Firebase



## Deployment

To run this project in your local system 

```bash
  git pull https://github.com/deepayannandy/nodeBackend.git
  cd nodeBackend
  npm i
  npm run devStart
```
